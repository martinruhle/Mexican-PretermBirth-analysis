# Installation Guide

Complete step-by-step instructions for setting up the analysis environment.

## System Requirements

- **R:** Version 4.4.2 or higher
- **RStudio:** Recommended for interactive analysis
- **RAM:** 8 GB minimum, 16 GB recommended

## Quick Start
```bash
# Clone repository
git clone https://github.com/martinruhle/Mexican-PretermBirth-analysis.git
cd Mexican-PretermBirth-analysis

# In R console
install.packages("renv")
renv::restore()
```
